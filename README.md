# Ashita4-HitboxDisplay
Shows various hitboxes and distance circles around entities. 

Vibe-coded so I could have cool-looking screenshots when documenting [FFXI range/radius mechanics](https://sruonlsb.notion.site/FFXI-Range-Radius-testing-2b23768470138076bf5cfa5020e375bc#2ba37684701380cab0a3d0a12fbee7fa)

Not intended as a meaningful FFXI addon, **will not be supported**.
